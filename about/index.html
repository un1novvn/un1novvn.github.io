03年生，在广东惠州某山卡拉小镇长大。以下是我的简要经历：

  • 2010 - 2016 小学
  • 2016 - 2019 初中，小镇做题家初级形态
  • 2019 - 2022 高中，小镇做题家完全形态
  • 2022 - 至今 就读于广东某双非本科

小学初中期间，没有自己的电脑，所以能碰电脑的机会都用来打游戏。高中正好赶上新冠疫情，上网课要电脑，就摸鱼学 java。
最开始是看b站动力节点的课，从怎么使用 cmd 开始看起。由于高中大部分时间都在学校，所以只有寒暑假能学。
三年一路看下来， 从 java 基础语法，到 mysql， 到 tomcat，到前端，算得上是计算机启蒙了，大学自然而然选计算机专业。

大学入学，开始接触 CTF，加入校队，开启CTF选手生涯。后来的生活，大都在本博客有记录了。

关于跑步：
从2016年开始，当时只是为了准备体育中考，只跑1千米。
体育中考结束，高中开始中长跑，4-6公里。高三运动会1500米，4分58秒，历史最好成绩了。
大学，开始接触长跑：10公里，15公里，半马，全马。

为啥我的 id 叫 unknown ? 我只记得最开始是小学时候听到一首很好听的歌，歌手那一栏是 unknown（后来才知道只是因为歌很冷门， 找不到歌手而已），歌一直听，这个单词印象越来越深，而且看它也长得挺清秀的，就拿来做id了，一用大概快10年了吧。
我的每个社交媒体基本都是这个id，有的平台已有这个id了，就在这个单词的基础上做一些别的变换，比如: unkn0wn, un1nown, unknovvn, 或者直接取前3位unk。
